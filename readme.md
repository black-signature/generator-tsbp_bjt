# Small / Medium Scale Typescript Project Boilerplate - yeoman generator

> Scaffolds out [SLDS Boilerplate] using yeoman


## Install

```
$ npm install generator-tsbp_bjt -g
```


## Usage

Run it with [`yo`](https://github.com/yeoman/yo):

```
$ yo tsbp_bjt
```
